# PingOne Platform APIs

The PingOne Platform is a comprehensive set of cloud identity and access management services to deliver innovative identity solutions for your enterprise. The PingOne APIs provide the tools to automate tasks, integrate with other platforms, and extend the functionality of the PingOne Platform.

## New to PingOne?

If you're new to PingOne, the following sections help you learn more and start working with PingOne APIs.

* [Getting Started](getting-started/introduction.html)

  Set up PingOne application connections and execute initial resource server workflows.

* [Before You Begin](before-you-begin/postman-and-pingone/download-the-pingone-postman-collections.html)

  Learn how to download and use the PingOne Platform API Postman collections.

* [Foundations](foundations/introduction.html)

  Learn about the PingOne SSO and Auth services with architectural descriptions.

* [Use Case Library](workflow-library/introduction.html)

  PingOne use case workflows as Postman collections you can use in your workspace.

* [Native SDKs](native-sdks/introduction.html)

  Build with PingOne Native iOS and Android SDKs for MFA, Verify, Wallet, and Protect to interact with the Platform API.

## PingOne SSO and Authorization Services

The PingOne Platform SSO APIs provide the foundation for managing identities, applications, and configurations. The Platform Auth APIs provide centralized authorization and authentication services to confirm a user's identity.

* [PingOne SSO (Single Sign-On) APIs](platform/introduction.html)

  Enable key administrator, application, and identity management capabilities. The core of the PingOne service.

* [Platform Auth APIs](auth/introduction.html)

  Centralize authorization and authentication to confirm user identities by orchestrating flows into configurable auth workflows.

## PingOne Universal Services

PingOne Universal Services are independent SaaS components that add pre-built identity security and verification to any application. They integrate seamlessly with PingOne, other Ping products, or third-party platforms.

* [PingOne Authorize](authorize/introduction.html)

  Handle authorization decisions for fine-grained access control.

* [PingOne MFA (Multi-Factor Authentication)](mfa/introduction.html)

  Secure CIAM and workforce logins using various methods such as biometrics and OTPs.

* [PingOne Protect](protect/introduction.html)

  Centralize threat protection against account takeover and fraudulent sign-ups.

* [PingOne Credentials](credentials/introduction.html)

  Manage and verify digital verifiable credentials for secure digital wallet experiences.

* [PingOne Verify](verify/introduction.html)

  Verify user identities using government IDs, facial comparison, and selfies.

* [PingOne DaVinci](davinci/introduction.html)

  Design and automate complex identity journeys using a visual flow builder (orchestration).
