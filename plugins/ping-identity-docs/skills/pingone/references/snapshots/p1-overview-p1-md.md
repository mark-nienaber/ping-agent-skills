# Overview page

The **Overview** page has two sections. The top part of the page shows an overview of sign-on activity for the selected time period.

![A screen capture of the overview dashboard](_images/ciy1646249996311.png)

The **Services** section shows a list of current services in the environment, and an option to [add services](settings/p1_add_a_service.html) to your current PingOne environment.

![A screen capture of the Services section of the Overview page.](_images/qxu1646249127194.png)

You can find detailed information about a service by viewing the corresponding [dashboard](monitoring/p1_dashboards.html) for the specific service.
