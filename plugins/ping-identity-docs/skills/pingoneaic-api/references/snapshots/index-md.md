# PingOne Advanced Identity Cloud API

PingOne Advanced Identity Cloud is a SaaS-based identity and access management (IAM) solution designed for workforce, consumer, and B2B identities. The Advanced Identity Cloud REST API provides full lifecycle control over identities, access policies, and the cloud environment itself. It allows developers and administrators to interact with the platform without using the web-based admin console.
