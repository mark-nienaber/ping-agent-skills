# Which PingOne am I using?

This guide can help determine whether you are using PingOne or a version of PingOne for Enterprise, so that you can access the appropriate documentation and support.

## PingOne or PingOne for Enterprise?

The PingOne Platform and PingOne for Enterprise are different products that use separate interfaces.

The best way to determine which product you're using is the URL that you use to access the administration portal.

| Product                | Admin Console URL   | Documentation Link                                                                          |
| ---------------------- | ------------------- | ------------------------------------------------------------------------------------------- |
| PingOne                | console.pingone.com | [PingOne](https://docs.pingidentity.com//pingone/p1_cloud__platform_main_landing_page.html) |
| PingOne for Enterprise | admin.pingone.com   | [PingOne for Enterprise](pingone_for_enterprise/p14e_overview.html)                         |

There are four versions of PingOne for Enterprise with different capabilities to meet different use cases.

You can identify which version of PingOne for Enterprise you're using by looking at the top navigation bar.

## PingOne for Enterprise

The top navigation bar on PingOne for Enterprise has menus for **Dashboard**, **Applications**, **Users**, **Setup**, and **Account**.

![Screen capture of the PingOne for Enterprise top navigation menu](_images/gak1643670747185.png)

Learn more in the [PingOne for Enterprise documentation](pingone_for_enterprise/p14e_getting_started.html).

## PingOne for Enterprise for Managed Service Providers

PingOne for Enterprise for Managed Service Providers has an additional **Customers** menu on the right of the top navigation bar.

![Screen capture of the PingOne for Enterprise for Managed Service Providers top navigation menu](_images/nki1643670820159.png)

Learn more about managing customer accounts with PingOne for Enterprise for Managed Service Providers in [Monitoring and modifying customer accounts](pingone_for_enterprise/p14e_monitoring_modifying_customer_accounts.html)

## PingOne SSO for SaaS Apps

PingOne SSO for SaaS Apps has an additional menu for **Customer Connections** between **Applications** and **Setup** on the top navigation bar.

![Screen capture of the PingOne SSO for SaaS Apps top navigation menu](_images/wpj1643670883556.png)

Learn more in the [PingOne SSO for SaaS Apps documentation](pingone_sso_for_saas_apps/p14saas_getting_started.html).

## PingOne SSO for SaaS Apps with Managed Accounts

In addition to the **Customer Connections** menu, PingOne SSO for SaaS Apps with Managed Accounts has a **Managed Accounts** menu on the right of the top navigation bar.

![Screen capture of the PingOne SSO for SaaS Apps for Managed Accounts top navigation menu](_images/uzk1643670953900.png)

Learn more in [Manage partner accounts](pingone_sso_for_saas_apps/p14saas_manage_partner_accounts.html).
