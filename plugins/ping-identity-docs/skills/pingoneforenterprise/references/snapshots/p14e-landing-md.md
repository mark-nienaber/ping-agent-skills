# PingOne for Enterprise

PingOne for Enterprise is a cloud solution that lets you connect your workforce users to any SAML or OpenID Connect (OIDC) application with a single sign-on (SSO).

See [Ping Identity Services](https://status.pingidentity.com/) for the operational status of all Ping Identity services, including PingOne for Enterprise. You can also choose to subscribe to alert notifications.

You can also connect to the following heartbeat endpoints to monitor PingOne for Enterprise operational status:

* SSO: <https://sso.connect.pingidentity.com/sso/heartbeat>

* Admin Portal: <https://admin.pingone.com/web-portal/heartbeat>
