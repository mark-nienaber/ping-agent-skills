# Configuration Guide Overview

Ping Identity supports standards-based identity and access management (IAM) integrations with a wide range of business applications.

These configuration guides cover SAML single sign-on for PingFederate and PingOne. To search and discover more integrations, find downloads, and read documentation, visit the Ping Identity [Integration Directory](https://support.pingidentity.com/s/marketplace-integration-home-page).
