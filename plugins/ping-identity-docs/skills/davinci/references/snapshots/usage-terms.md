# DaVinci browser support

PingOne DaVinci is supported on the following platforms.

|   |                                                            |
| - | ---------------------------------------------------------- |
|   | Scripts or custom CSS included in flows may limit support. |

| Browser                | Version                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Google Chrome          | Three most recent major versions                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Mozilla Firefox        | Three most recent major versions                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Apple Safari for MacOS | Two most recent major versions                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Microsoft Edge         | Two most recent major versions                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Internet Explorer      | Any version using Trident 7+ or EdgeHTML 18+ WebView rendering engines.Internet Explorer support is limited, and is intended for situations where upgrading to a Chromium browser is not yet possible.You must enable Internet Explorer compatibility mode on a per-flow basis. Some features, such as PingOne Forms, are not supported in this mode. Learn more about enabling this feature in [Editing flow settings](../flows/davinci_editing_flow_settings.html). |
