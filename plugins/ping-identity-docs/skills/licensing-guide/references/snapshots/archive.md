# Archive

The archive section contains licensing components that are no longer offered to net-new customers.

## Advanced identity software

**Governance:** Identity governance capabilities (access request, access review and certifications, segregation of duties (SoD), and role modeling) that manage who has what access, ensure compliance, and drive lifecycle-aligned access decisions.

| Feature        | Description                                                                                                                                              |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Access Review  | Identity governance capability for running access certifications, reviewing who has what access, and remediating inappropriate access to stay compliant. |
| Access Request | Identity governance capability for requesting, approving, and tracking access to applications, roles, and entitlements using catalog-driven workflows.   |
