# PingOne Recognize

PingOne Recognize helps businesses authenticate users through privacy-preserving facial biometrics.

![PingOne Recognize overview diagram showing the four core functional capabilities: enrollment, authentication, account recovery, and workforce.](introduction/_images/p1recognize-overview.png)

With one glance at a device's camera, the technology provides multi-factor authentication in milliseconds and ensures that only the person who originally enrolled can authenticate.

PingOne Recognize calls this genuine identity assurance.
