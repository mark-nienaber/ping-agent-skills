# User Actions

In addition to the core SCIM actions (Create, Read, Update, Delete), PingOne for Enterprise Directory also supports the following actions for users:

* User authentication: `/content/developer/en/api/pingone-directoryapi.html?wcmmode=disabled#user_authentication`

* Invitation of new users to your directory: `/content/developer/en/api/pingone-directoryapi.html?wcmmode=disabled#user_invitation)`
